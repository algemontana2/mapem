#  MapEm

MapEm is a genealogy mapping tool that takes GEDCOM data and visualizes migration patterns across time using a Flask backend and a React frontend.

## Features
- GEDCOM upload and parsing
- Smart location geocoding (historical + vague support)
- Animated migration maps with filters
- Manual override + logging of unresolved places

## Stack
- Python (Flask) backend
- React + Vite frontend
- PostgreSQL DB
- OpenAI CLI integrations
